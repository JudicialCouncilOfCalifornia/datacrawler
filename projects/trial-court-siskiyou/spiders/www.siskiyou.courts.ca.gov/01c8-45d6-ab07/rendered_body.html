CASE RECORDS REQUEST

If you need copies from your case file, please open the form, print it, then complete it and return to the address on the form. The Court will process and return your copies, provided you enclosed the fees and the self-addressed, stamped envelope.

For a Criminal/Traffic Case, mail request to:
Criminal Division – Records Request
311 4th Street, Room 206
Yreka, CA 96097

For all other cases, mail request to:
Civil Division – Records Request
311 4th Street, Room 206
Yreka, CA 96097
