Do It Yourself

We have a document preparation program available for you to use to complete the necessary forms to file an Unlawful Detainer/Eviction Complaint or Answer. This program will ask you to answer questions that are needed to file. The answers you give will be used to complete the forms needed to file. This program will also allow you to print out all the completed necessary forms. Do It Yourself- Complaint
